## 🤖 What is Machine Learning?  

Machine Learning (ML) is a way for computers to **learn patterns from data** and make decisions without explicit programming.  

> 💡 Instead of writing rules → we provide data → the model learns  

### 🔍 Examples  
- Face ID unlocking your phone  
- Netflix recommendations  
- Instagram feed personalization  

---

## 🎯 Types of Machine Learning  

### 🧑‍🏫 Supervised Learning  
![Supervised](https://img.shields.io/badge/Data-Labeled-success)

- Uses **labeled data**  
- Learns input → output mapping  

#### 📌 Examples  
- Email → Spam / Not Spam  
- Image → Cat / Dog  

> 🧠 Think: Learning with a **teacher + answer key**  

---

### 🕵️ Unsupervised Learning  
![Unsupervised](https://img.shields.io/badge/Data-Unlabeled-blueviolet)

- Uses **unlabeled data**  
- Finds hidden patterns automatically  

#### 📌 Examples  
- Customer segmentation  
- Grouping similar users  

> 🧠 Think: A **detective finding patterns**  

---

## ⚠️ Overfitting (Intuition Only)  

![Concept](https://img.shields.io/badge/Concept-Overfitting-red)

Overfitting happens when a model **memorizes data instead of learning patterns**  

### ❌ Problem  
- Performs well on training data  
- Fails on new data  

### 🎯 Analogy  
> A student memorizing answers instead of understanding concepts
